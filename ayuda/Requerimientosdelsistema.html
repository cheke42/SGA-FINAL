Interfaces de hardware


Cualquiera de los equipos informáticos en que se desee instalar el SGA, deberá estar en perfecto estado y  contar con las siguientes características: Procesador de 1.66GHz o superior, memoria mínima de 512Mb, mouse, teclado y monitor. El Aquarium deberá contar con una red local para los equipos en que se desee instalar el SGA.


Interfaces de software


La comunicación entre los módulos del sistema se realizará mediante una base de datos centralizada, alojada en un servidor.
